# Отчёт по анализу текущего процесса (AS-IS)

> **Процесс:** Обработка и выполнение заявки на перевозку  
> **Владелец процесса:** Директор по операциям
---

## 1.1. Характеристика процесса

| Параметр | Значение |
| :--- | :--- |
| **Название процесса** | Обработка и выполнение заявки на перевозку |
| **Владелец процесса** | Директор по операциям |
| **Участники (роли)** | Клиент, Диспетчер, Менеджер по продажам, Руководитель, Водитель, Бухгалтер |
| **Триггер** | Поступление заявки от клиента |
| **Результат** | Доставленный груз, оплаченный счёт |
| **Смежные процессы** | Управление транспортом, Бухгалтерский учёт, CRM |

## 1.3. Три критичные проблемы


1. **Потеря заявок (19%)**
 Напрямую режет выручку компании, тормозит захват рынка и сливает лиды.
2. **Время подтверждения заявки — 3,5 дня**
Из-за долгой работы с документами, клиенты может уйти к конкурентам.
3. **Отсутствие уведомлений о статусе груза**
Из-за этого диспетчерам обрывают телефоны — **68% всех звонков** идут только ради вопроса где находится груз.
